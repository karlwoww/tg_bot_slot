# 🎰 Slot Machine Bot

Telegram-бот с виртуальной слот-машиной, написанный на **Python** с использованием **aiogram 3.x**. Игроки могут делать ставки, крутить барабаны, соревноваться за место в рейтинге и управлять виртуальным балансом.

## ✨ Основные возможности

* 🎰 Игровой автомат на основе Telegram Dice (`🎰`)
* 💰 Виртуальный баланс для каждого пользователя
* 🏆 Рейтинг игроков по размеру баланса
* 🛡️ Административная панель управления
* ⏳ Защита от спама с помощью кулдауна
* 📊 Хранение данных в SQLite

---

## 🎯 Выигрышные комбинации

| Комбинация | Описание      | Множитель | Условие               |
| ---------- | ------------- | --------- | --------------------- |
| 🎰🎰🎰     | Джекпот (777) | x50       | `value == 64`         |
| 🍋🍋🍋     | Три лимона    | x25       | `value == 43`         |
| 🍇🍇🍇     | Три винограда | x15       | `value == 22`         |
| 🔔🔔🔔     | Три барабана  | x10       | `value == 1`          |
| ❌          | Проигрыш      | -1 ставка | любое другое значение |

> Значение `value` получается из результата Telegram Dice:
>
> ```python
> await bot.send_dice(chat_id, emoji="🎰")
> ```

## 🤖 Пользовательские команды

### Основные команды

| Команда  | Описание                                   |
| -------- | ------------------------------------------ |
| `/start` | Регистрация пользователя и главное меню    |
| `/myid`  | Показать Telegram ID                       |
| `/admin` | Открыть админ-панель (для администраторов) |

### Главное меню

| Кнопка         | Действие                |
| -------------- | ----------------------- |
| 🎰 Слот-машина | Начать игру             |
| 💰 Баланс      | Показать текущий баланс |
| 🏆 TOP         | Топ-5 игроков           |

---

## 🎮 Игровое меню

| Кнопка             | Действие                            |
| ------------------ | ----------------------------------- |
| Выбор ставки       | Выбор суммы ставки от 8 до 10 000 ₽ |
| GO                 | Запустить слот-машину               |
| Поменять ставку    | Вернуться к выбору ставки           |
| Информация о слоте | Показать правила и комбинации       |
| Выйти              | Вернуться в главное меню            |

### Защита от спама

Кнопка **GO** имеет кулдаун **10 секунд** между нажатиями для каждого пользователя.

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/karlwoww/tg_bot_slot.git
cd tg_bot_slot
```

### 2. Создание виртуального окружения

#### Linux / macOS

```bash
python -m venv venv
source venv/bin/activate
```

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
TOKEN=YOUR_BOT_TOKEN
ADM_IDS=YOUR_TELEGRAM_ID
```

### 5. Запуск бота

```bash
python main.py
```
